register_globals

register_globals 是php.ini 中的一个配置，该配置直接影响PHP 如何接收和传递参数。

我的理解：

PHP存在注入$_GET、$_POST等系统变量（array）,当注册register_globals后其包含的大量系统数组变量直接释放到直接变量中如$_ENV[“os”]==> $os (测试失败了，未果)
